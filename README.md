# Nova Memory Match
A JavaScript memory match application, all devices oriented.
## Features
- Shuffle cards for new game.
- A timer to record time spent for one game.
- User can pause game.
- User can continue game.
## Technologies
- JavaScript
- jQuery
- HTML5
- CSS3
- Bootstrap 4
